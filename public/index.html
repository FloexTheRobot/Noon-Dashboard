Auto Creation

Create from SKU → noon

Enter an SKU. Its description, images & details are fetched from connect.oskarme.com and the product is created on noon via the Partner API — fully automated, no manual upload.

Fallback — NIS catalogue file

Export all created products as a noon-ready NIS Excel file (manual Seller Lab upload), or push it via the saved browser session.

Mode 3 — pricelist → NIS (single + variations)

Upload a supplier pricelist (.xlsx) (columns like Product Description, Features, SKU, Barcode, Color…). Variants with a merged description cell are grouped automatically (color → size variation), section headers like HEADPHONE/POWER BANK drive classification, and images are pulled from connect per SKU. One NIS file is generated for every SKU.
